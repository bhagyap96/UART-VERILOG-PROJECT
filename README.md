# UART Verilog Project

## Description
This project implements a UART transmitter and receiver using Verilog HDL.

## Features
- FSM-based TX and RX design
- Baud rate generator
- Verified using simulation testbench

## Skills Used
- Verilog HDL
- Digital Design
- FSM Design
- UART Protocol

## Tools
- ModelSim / Vivado
